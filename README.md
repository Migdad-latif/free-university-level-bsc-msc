# 🎓 Open DS & FinTech University

**A free, open-source, self-paced curriculum equivalent to a full BSc + MSc in Data Science with Financial Technology specialisation.**

No tuition fees. No admissions process. No gatekeeping. Just 28 structured modules, free resources, and real portfolio projects - built for anyone, anywhere in the world.

---

## What is this?

This is a complete 5-year self-study curriculum that mirrors the structure and rigour of a university BSc + MSc in Data Science and FinTech. Every module links directly to free, open-access learning resources - MIT OpenCourseWare, fast.ai, Coursera (audit), Kaggle, Harvard CS50, free textbooks, and more.

The curriculum is delivered as a single interactive HTML file you can open in any browser. No installation required. No account needed. Works offline once loaded.

**Live demo:** open `open_ds_fintech_university.html` in your browser.

## Curriculum Structure

### 🔵 BSc - Years 1 to 3 (18 modules)

| Year 3 | FinTech Specialisation & Capstone | Deep Learning, NLP for Finance, Risk Modelling, Algorithmic Trading, RegTech & Compliance, BSc Capstone Project |

| Year | Focus | Modules |
|------|-------|---------|
| MSc Year 1 | Core MSc Methods | Big Data Analysis (DSM010), Data Programming in Python (DSM020), Statistics & Statistical Data Mining (DSM030), Machine Learning (DSM040), Financial Data Modelling (DSM120) |
| MSc Year 2 | FinTech Specialist + Dissertation | Blockchain Programming (DSM070), Financial Markets (DSM080), Neural Networks (DSM150), Natural Language Processing (DSM140), Final Project (DSM500) |

## Features

- **28 fully detailed modules** - each with topics covered, free resources, learning outcomes, and a portfolio project
- **Clickable resource cards** — every resource links directly to the free material (MIT OCW, fast.ai, Kaggle, free textbooks, YouTube, Coursera audit etc.)
- **Progress tracking** — mark modules complete; progress saves to your browser
- **No external dependencies** — runs entirely in-browser, no internet required after first load
- **Mobile friendly** - works on any screen size
- **Free learning platforms panel** — quick access to MIT OCW, Kaggle, fast.ai, freeCodeCamp, Hugging Face, 3Blue1Brown, StatQuest, arXiv, Harvard CS50, Khan Academy, and more

## How to Use

### Option 1: Open directly
```
git clone https://github.com/Migdad-latif/free-university-level-bsc-msc.git
open open_ds_fintech_university.html
```

### Option 2: GitHub Pages (recommended)
1. Fork this repository
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your curriculum will be live at `https://Migdad-latif.github.io/free-university-level-bsc-msc/`

### Option 3: Download and open
Click the green **Code** button → **Download ZIP** → extract → open `open_ds_fintech_university.html` in your browser.

## Portfolio Projects

Every module includes a portfolio project designed to be published on GitHub. By the end of Year 3 (BSc level) you will have built:

- A financial data pipeline (Python, Pandas, APIs)
- A credit default prediction model (XGBoost, SHAP)
- A stock price volatility forecaster (ARIMA, GARCH, LSTM)
- A portfolio optimiser (Markowitz efficient frontier)
- A fraud detection system (autoencoders, PyTorch)
- A real-time transaction processing pipeline (Kafka, PySpark)
- An ML model deployed as a production API (FastAPI, Docker)
- An earnings call sentiment analyser (FinBERT, Streamlit)
- A credit risk scorecard with IFRS 9 ECL (SHAP, regulatory report)
- A systematic trading strategy backtest

By the end of Year 5 (MSc level) you will also have completed a full MSc-standard final project (DSM500) - a 8,000–10,000 word report with deployed code.
